# Novera Product Logic

Novera is a cultural ecosystem platform for artists, events, creative spaces, collectives, live rooms, open calls, AI matching and city-based cultural discovery.

Core manifesto:

Discover.
Create.
Collaborate.
Beyond.

## Main sections

- Home
- Explore
- Artists
- Events
- Creative Map
- Live Rooms
- Spaces
- Collectives
- Open Calls
- AI Match
- Activity

## Main models

### User
Represents a registered user.

### Artist Profile
Represents the creative identity of a user.

### City
Represents a cultural city ecosystem.

### District
Represents a creative area inside a city.

### Event
Represents cultural events, screenings, performances and exhibitions.

### Space
Represents venues, studios, galleries and creative spaces.

### Collective
Represents creative groups and communities.

### Live Room
Represents live audio/listening rooms.

### Open Call
Represents collaboration opportunities.

### Activity
Represents ecosystem activity feed items.

### AI Match
Represents personalized creative recommendations.
